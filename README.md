# Galton Board Simulator (高尔顿板模拟器)
这是一个基于 Python 的交互式高尔顿板（豆机）模拟程序。它通过模拟小球在钉子板间的物理碰撞与下落过程，直观地展示了二项分布如何逼近正态分布（中心极限定理）。   
   
### 本项目不仅仅是一个数学演示，更是一个物理实验室，允许用户探索物理惯性对统计分布的影响。   

## ✨ 主要功能
实时物理/逻辑混合模拟：结合物理重力、空气阻力与二项分布逻辑，模拟小球下落、反弹及堆积过程。

动态数据可视化：左下角实时绘制小球落点直方图，并叠加理论正态分布曲线（PDF），实时计算并对比理论方差与实际方差。

智能堆积渲染：采用“动态垂直压缩算法”，无论生成多少小球（几百或几千），收集框内的堆积效果均能自动缩放，保证不溢出屏幕且视觉美观。

交互式探索：

- Total Balls：自定义小球总数。

- Rows：自定义钉子板层数。

- Gravity/Speed：调节下落速度。

- 拟合度反馈：系统会自动评估当前分布与理论值的吻合度，并给出 "Perfect Fit" 或 "High Inertia" 等状态提示。

## 🛠️ requirements
- python 3.12
- 运行前请确保安装以下 Python 库：

```Bash
pip install pygame matplotlib scipy numpy
```
   
## 🚀 运行方法
直接运行主程序文件：

```Bash
python GaltonBoardSim.py
```
   
   

## 📝 核心逻辑说明
程序没有强制修正小球的物理轨迹以符合数学模型，而是保留了可间接调试的物理惯性。   
思考：可供用户调节的三个参数与正态分布成立存在什么关系？为什么？   

- 参考调节参数：`1000, 12, 0.285`
